When a nested dialog is opened from the About dialog, it is no longer possible to close the About dialog using the "X" button. To test:

  1. 1. Click "About" to open the dialog
  2. 2. Click "OPEN NESTED" inside the dialog to open a nested dialog.
  3. 3. Close the nested dialog
  4. 4. At this point, the "X" button in the About dialog does nothing. The only way to close the dialog is by pressing the Esc key.
